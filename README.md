# 🎬 Video Sync — расширение для совместного просмотра видео

Браузерное расширение (Chrome / Edge / Brave), которое позволяет смотреть видео вместе с друзьями в реальном времени — на любом сайте. Воспроизведение, паузы и перемотка синхронизируются у всех участников комнаты мгновенно.

**Работает на:** YouTube, HDrezka, Кинопоиск, Netflix, Twitch, Plex, а также на любом другом сайте с `<video>`-плеером — в том числе со встроенными iframe-плеерами.

---

## ✨ Возможности

- 🔄 Синхронизация воспроизведения, паузы и перемотки в реальном времени
- ⚡ Синхронизация скорости воспроизведения (0.25× — 3×)
- 💬 Встроенный чат комнаты
- 🎭 Поддержка iframe-плееров (видео загружено во встроенном фрейме)
- 🔗 Уведомление о переходе к новой серии (опционально, без авто-редиректа)
- 👥 Список участников в реальном времени
- 🔁 Автоматическое переподключение при обрыве связи
- 🌐 Работает на любом сайте — не требует настройки под конкретный ресурс

## 📦 Установка расширения

1. Скачайте или склонируйте репозиторий
2. Откройте в браузере: `chrome://extensions`
3. Включите **«Режим разработчика»** (переключатель вверху справа)
4. Нажмите **«Загрузить распакованное расширение»**
5. Выберите папку `extension/` из этого репозитория
6. Расширение появится на панели инструментов браузера

> Протестировано в Chrome 124+, Edge 124+, Brave.

## 🚀 Запуск сервера

Серверная часть — это Node.js-приложение в корне репозитория. Вы можете запустить его бесплатно в облаке или локально.

### ☁️ Railway (бесплатно, рекомендуется)

1. Зарегистрируйтесь на [railway.app](https://railway.app)
2. Нажмите **New Project → Deploy from GitHub repo**
3. Подключите этот репозиторий
4. Railway автоматически обнаружит `package.json` и запустит `npm start`
5. В разделе **Settings → Networking** включите публичный домен
6. Скопируйте URL вида `https://xxx.up.railway.app`
7. В расширении нажмите **✏️ Изменить** и введите: `wss://xxx.up.railway.app`

### ☁️ Render (бесплатно)

1. Зарегистрируйтесь на [render.com](https://render.com)
2. **New → Web Service** → подключите репозиторий
3. **Build Command:** `npm install`
4. **Start Command:** `node server.js`
5. Скопируйте URL вида `https://xxx.onrender.com`
6. В расширении укажите: `wss://xxx.onrender.com`

> ⚠️ На бесплатном тарифе Render сервер засыпает после ~15 минут бездействия.  
> Первое подключение может занять 30–60 секунд («пробуждение» сервера).

### 🖥️ Свой сервер / локально

```bash
# Установить зависимости
npm install

# Запустить
npm start

# Режим разработки (с авто-перезапуском)
npm run dev
```

Сервер по умолчанию запустится на порту `3000`. Для продакшена задайте переменную окружения `PORT`.

В расширении укажите адрес своего сервера, например: `ws://localhost:3000`

## 🎮 Как пользоваться

1. Откройте сайт с видео (YouTube, HDrezka, Кинопоиск и т.д.)
2. Нажмите на иконку расширения в панели браузера
3. Введите своё имя
4. Создайте комнату (`+ Создать`) или введите код и нажмите `Войти`
5. Поделитесь 6-значным кодом комнаты с другом
6. Друг вводит тот же код — и вы смотрите синхронно 🎉

Все действия с плеером (воспроизведение, пауза, перемотка) мгновенно применяются у всех участников.
